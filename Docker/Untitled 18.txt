それを起動


クライアント
docker run


docker img runでイメージを作成して、、webrrick:lateset でイメージ作ってますね。

コンテナの作成と起動は
docker container run  sample/webrick:lateset <-名前やってね？w


docker container stop "コンテナ名"　--コンテナストップ
docker container rm 'コンテナ名'　　ーー削除
docker container ls --起動中のコンテナを表示(-a で停止中も表示
docker container logs 'コンテナ名'　--コンテナの履歴を見れる。


docker container exec webrick ruby -v
