 

# model_nameとhuman

Railsが用意するメソッドです。

`**model_name**` メソッドは、モデルクラスに対して呼び出されると、そのモデルに関連する `**ActiveModel::Name**` オブジェクトを返します。このオブジェクトは、モデルの名前に関する様々なメソッド（例えば `**human**`、`**singular**`、`**plural**` など）を提供します。

例えば、`**User.model_name.human**` というコードは、`**User**` モデルの「人間に優しい」名前を返します。これは通常、i18n ファイルで定義された翻訳に基づいており、ユーザーインターフェースでの表示に適しています。

あなたのコード例では、`**Roulette.model_name.human**` が使われています。これは `**Roulette**` モデルの人間に優しい名前を返すことを意味し、おそらく ActiveAdmin のメニュー項目の表示名として使用されています。これにより、コード内でハードコーディングされた文字列を使用する代わりに、翻訳ファイルを介して柔軟に名前を管理できます。

ざっくりとした理解としては、辞書ファイルを参照するようになる。
