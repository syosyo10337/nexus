 

# FileとBlobの違い

**Blob（ブロブ） Binary Large Object**

- **特徴:**生のバイナリデータ（データのかたまり）を格納します。

- **用途:**メタデータが不要な、一般的なバイナリデータ（画像、動画、音声など）を扱う際に使用されます。

- **例:**Web開発において、画像などの大容量データを扱う際に利用されることがあります。

**File（ファイル）**

- **特徴:**
    
    `Blob`インターフェースを継承しており、`Blob`が持つ特性に加え、`name`（ファイル名）や`lastModified`（最終更新日時）といったファイル特有のメタデータを持つことができます。
    

- **用途:**
    
    ユーザーがアップロードしたローカルファイルなど、ファイル名を持つファイルを扱うのに適しています。
    

- **例:**
    
    HTMLの`<input type="file">`要素で取得できるファイルが`File`オブジェクトです。
    

つまり、 `File` は特殊 `Blob` オブジェクトであります。

File extends Blob
