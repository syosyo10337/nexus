# イベントとは

---

ボタンが押されたり、ページが読み込まれたり、ビデオを再生したりといったブラウザーで起きることで、JavaScriptはそれに反応してコードを実行させることができます。

## **イベントリスナー**

イベントが発生したことを聞き取る機能

## **イベントハンドラー**

発生したイベントに反応して実行されるコードブロック

```JavaScript
//ボタンについたclass属性を参照し
// guessSubmitという定数にそのHTMl要素を代入
const guessSubmit = document.querySelector('.guessSubmit')

//その定数(HTML要素)に対して、イベントリスナーを追加
guessSubmit.addEventListener('click', checkGuess);
```

## addEventListener()

```JavaScript
//基本書式
EventTarget.addEventListener(type, listner);

e.g.)
//checkGuess()は関数
//(addEventListener()の中では括弧を書く必要はない。)
guessSubmit.addEventListener('click', checkGuess);
```

第一引数に,イベントの種類を取り、第二引数にイベントが発生した場合に実行するコードを取る。
