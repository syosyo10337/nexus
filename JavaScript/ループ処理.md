 

# ループ処理

---

ループ処理の多くは、次のような機能を持ちます。

**カウンター:**  
 ループの開始地点で、初期化される値です

**条件:**  
 ループの実行を継続するか終了するかを決める true/false の判定です。たいていはカウンターがある値に達した場合に終了します。

**イテレーター:**  
 これは一般的には条件が `true`  
. では無くなるまで、カウンターの値をループごとに少量ずつ増加させます。

# for文

---

```JavaScript
// 基本書式
// for ()の括弧内は;で区切ること。
// カウンターの変数は慣習的にiがよく使われる。(カウンター変数やイテレータ変数とも呼ばれる)
for (初期化式(変数の定義)); 条件式; 変化式){
  実行する処理;
}

//e.g.
for (let i = 0; i < 10; i++) {
  console.log(`hello ${i}`);
}
```

制御の流れは、

①一度初期化式を実行する。

②条件式を評価してtrueが返された場合にはブロック内の処理を実行し

③変化式を一度実行する。

### breakステートメント

---

==すべての繰り返し処理が終了する前にループを終了==したいとき、[break](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/break)  
 文を使用して終了させることができます。

### continue ステートメント

---

[continue](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/continue)文は `break` と同じような動作をします。けれど、ループを完全に抜けてしまうのではなく、次の繰り返しまで飛ばします。  

```JavaScript
//iが3の倍数のときのみ処理がスキップされて1~10まで出力する例
for(let i = 1; i <= 10; i++ {
	if(i % 3 === 0) {
	continue;
	}
	console.log(i);
}
```

	実行する処理;
	変化式//いずれloopが終了するようにする　
}
```

//e.g.)

let hp = 100;

	console.log(`${hp}HP left`1);
		hp -= 15

```

# do..while

---

一度do以下の処理を実行した後に、条件式の評価が行われる。条件に関わらず最低は一度は処理を実行したい時に用いる。

```JavaScript
//基本書式
初期化処理
do {
	実行する処理

	変化式
} while (条件式)
```

# for … in

---

配列のインデックスやオブジェクトのプロパティ等を処理できる.

```JavaScript
const array = [65, 55, 80, 100];

for(const index in array) {
  console.log(index);
}
// => 0
// => 1
// => 2
// => 3

for(const index in array) {
  console.log(array[index]);
}
// => 65
// => 55
// => 80
// => 100
```

```JavaScript
const obj = {name: 'soarflat', country: 'Japan'}

for(const key in obj) {
  console.log(key);
}
// => name
// => country
```

# for … of

---

配列の値に対して処理ができる。

```JavaScript
const array = [65, 55, 80, 100];

for(const value of array) {
  console.log(value);
}
// => 65
// => 55
// => 80
// => 100
```

### オブジェクトのプロパティを参照したい時

```JavaScript
const obj = {name: 'soarflat', country: 'Japan'}

for(const key in obj) {
  console.log(obj[key]);
}
// => soarflat
// => Japan
```
