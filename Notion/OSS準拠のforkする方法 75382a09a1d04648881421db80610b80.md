## 直接PRできないときに、fork先からPRを作成する方法

originを自分のremoteに、  
upstreamを本家のremoteにした状態で

```Bash
# localのマスタで,本家と同期する
$ git checkout master
$ git fetch upstream
$ git merge upstream/master --ff-only #optionは必要?

# origin(自分のリモートも同期させる。)
$ git push origin master -u #(upstream必要?)
```
