# CustomPainter

このクラスをextendsしたカスタムクラスのpaintメソッドで自由に描画することができる

```Dart
final customPaint = CustomPaint(
 painter: myPainter(),
 child: SomeWidget(),
);

class Mypainter extends CustomPainter {
 @override
 void paint(ui.Canvas canvas, ui.Size size) {
  //implement paint
 }

@override
bool shouldRepaint(CustomPainter oldDelegate)
 // implement shouldRepaint
 return null;
}
```

paint とshouldRepaintをoverrideする必要があります。

- paintは実際の描画ようのメソッド。

- shouldRepaintはbuildが走った時に再描画するか。

吹き出しを作る時。

実際に描画したい座標を指定するらしい。

```Dart
    var paint = Paint()
      ..color = const Color.fromARGB(255, 137, 172, 200)
      ..style = PaintingStyle.fill;

    var path = Path();
    // 吹き出しの形状をカスタマイズ
    path.moveTo(0, size.height);
    path.lineTo(size.width - 10, size.height);
    path.quadraticBezierTo(
        size.width, size.height, size.width, size.height - 10);
    path.lineTo(size.width, 10);
    path.quadraticBezierTo(size.width, 0, size.width - 10, 0);
    path.lineTo(10, 0);
    path.quadraticBezierTo(0, 0, 0, 10);
    path.close();

    canvas.drawPath(path, paint);
```

基本的にはcanvas.XXXメソッドを使って描画する。

sizeはpaintメソッドの引数として渡ってくるものの一つで、絶対値ではなく、親wigdetのサイズに依存した、相対値での指定ができます。
