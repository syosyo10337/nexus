 

# マルチカーソルを使う

✅ **Multi-cursor（マルチカーソル）によるデファクトスタンダードな方法：**

- `Ctrl+Alt+↓` / `Ctrl+Alt+↑`（Windows/Linux）または `Cmd+Option+↓` / `Cmd+Option+↑`（Mac）：縦方向に複数カーソルを作成

- `Alt+Click`（Windows/Linux）または `Option+Click`（Mac）：クリックした位置にカーソルを追加

- `Ctrl+D`（Windows/Linux）または `Cmd+D`（Mac）：選択中の単語と同じものを次々選択

- `Ctrl+Shift+L`（Windows/Linux）または `Cmd+Shift+L`（Mac）：選択したテキストのすべての出現箇所を一度に選択

## 複数コピー・複数貼り付け

✅ **標準的なアプローチ：**

1. **マルチカーソルでのコピー・貼り付け**：
    
    - 複数カーソルでテキストを選択
    
    - `Ctrl+C`でコピー
    
    - 別の場所で複数カーソルを作成し`Ctrl+V`で貼り付け
    

2. **複数クリップボードの活用**：
    
    - VSCode拡張機能「Multiple clipboards for VSCode」を使用
    
    - 複数の異なる内容を保存し、選択的に貼り付け可能
