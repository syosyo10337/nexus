# parameter と argument

---

## Parametersとは

変数(variables)のことで、特に、関数宣言をする際に定義するものを指す。

日本語役としては、仮引数と呼ばれる変数部分のこと

## Argumentとは

変数(varibles)の一種で、特に関数実行の際に、与えられる（もしくは渡される）ものを指す。

日本語では、実引数と呼ばれ、関数に渡される変数のこと。
