# 外部キー制約 (1)

### 外部キー（foreign key)

データベースの用語。あるテーブルにおいて、他のテーブルの主キーを参照する主キー以外のカラムのこと。  

### 外部キー制約(foreign key contrait)(foreign key: true)

親テーブルと子テーブルの2つのテーブル間でデータの整合性を保つために設定される制約。参照元(する側)が子テーブル、参照先(される側)が親テーブル

親テーブル(PKを持つ)が持たない値を子テーブル(FK)で持つことはできないという制約。

※外部キー制約 ⇒ FKがnot null では必ずしもない。
