## nginxの設定

nginxがRailsアプリと連携できるように設定する。

```Bash
# /etc/nginx/conf.d/rails.confファイルを"管理者権限を使って"作成して


upstream puma {
	# pumaの設定で指定することになる、socketファイルを指定する
	# recorda-meにはアプリのディレクトリ名をを入れましょう。
	server unix://var//www/recorda-me/tmp/sockets/puma.sock;
}

server {
	# nginxが待ち受けたいポートを指定
	listen 3000 default_server;
	listen [::]:3000 default_server;
	server_name puma;

	location ~^/assets/ {
		root /var/www/recorda-me/public;
	}

	location / {
		proxy_read_timeout 300;
		proxy_connect_timeout 300;
		proxy_redirect off;
		proxy_set_header Host $host;
	proxy_set_header X-Forwarded-Proto $h;
	server_name puma;
	
	location ~^/assets/ {
		root /var/www/recodra-me/public;
	}
	
	location / {
		proxy_read_timeout 300;
		proxy_connect_timeout 300;
		proxy_redirect off;
		proxy_set_header Host $host;
		proxy_set_header X-Forwarded-Proto $http_x_forwarded_proto;
		proxy_set_header X-Forwarded_For $proxy_add_x_forwarded_for;
		# server_name 
		proxy_pass http://puma;
	}
}
```

## Rubyのインストール

### ①rbenvのインストールを初期設定

```Bash
# rbenvをgithubから取得して
$ curl -fsSL https://github.com/rbenv/rbenv-installer/raw/HEAD/bin/rbenv-installer | bash

# rbenvコマンドを使うためにパスを通す
$ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile


# ?
$ source ~/.bash_profile
```

### ②Rubyをインストール

```Bash
#入れたいルビーのバージョンを指定してね 
#ec2-userのユーザディレクトリにいる時、installに失敗しがち 
$ rbenv install 2.7.6
$ rbenv global 2.7.6
```

### ③Railsをインストール

```Bash
e.g.)6.0.4をインストールする例
$ gem install rails -v 6.0.4
```

### ④アプリケーションファイルをクローン

```Bash
# ディレクトリを移動して
$ cd /var/www

$ git clone <URLとか>

#passwordにはアクセストークンを使うこと
```

### ⑤bundlerでRailsのライブラリを導入

```Bash
#ディレクトリを移動して
```

## RailsとRDSを関連づけるために、DBの初期設定をする

---

### mysqlの場合

①webサーバーからmysqlコマンドを使って、接続する

```Bash
# RDSの作成した時に作成したユーザ情報を元に接続する。
# -p パスワードを取る
# -u ユーザ名を指定
# -h ホスト名を指定
$ mysql -u <設定したユーザ名> -p -h <ホスト名>

e.g.)AWSのサンプルで作成したRDSの情報を元に接続する例

$ mysql -u admin -p -h db.home
```

②Mysqlにユーザとデータベースを作成する。

```SQL
# データベースとユーザを作成

myspl> CREATE USER sample_app identified by <パスワード作成>;

# sample_appデータベースの権限を今作ったユーザに委譲?
myspl> GRANT ALL PRIVILEGES ON sample_app.* TO sample_app@'%';
myspl>quit
```

```Bash
$ bundle exec pumactl start

$ pumactl start

$ bundle exec pumactl halt
```
