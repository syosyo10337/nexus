# プロパティ,定数,メソッドのアクセス権  
(visibility)

それぞれの値の可視性について以下のような種類で指定することができる。

- public

- private

- protected

## public

どこからでもアクセス可能です。

## private

private 宣言されたメンバーには、そのメンバーを定義したクラスからのみアクセスできます。(継承されたクラスからは参照できない。)

### protected

protected 宣言されたメンバーには、 そのクラス自身、そのクラスを継承したクラス、および親クラスからのみアクセスできる。

## Staticメソッド/プロパティ

静的メソッド/プロパティともいう。クラスのインスタンス化の必要なしにアクセスすることができるものをさす。

in PHP

`static`キーワードを用いて宣言する。

```PHP
class Some
{
	static function foo(){
	// do something
	}
}
```

クラスプロパティもしくはメソッドを static として宣言することで、 クラスのインスタンス化の必要なしにアクセスすることができます。 static として宣言されたプロパティやメソッドは、 インスタンス化されたオブジェクトの内部からも :: (スコープ定義演算子) を使ってコールできます。
