# 関数の定義

---

関数の定義には、原則3つの情報が必要になります。

- 関数名と出力のデータ型

- 引数とそのデータ型

- 関数の処理内容

==Rubyの場合には、関数ではなく、メソッドと呼ばれる。その所以は全てのものをオブジェクトとして扱うので、それに対する。メッセージとなるもの、つまりメソッド。という括りで対応できる。==

e.g.)底辺と高さをとって三角形の面積を計算する関数

```JavaScript
//JavaScriptでの関数定義
//JavaScriptは動的片付け言語なので、引数と出力のデータ型を指定する必要がない。

function triangleArea(width, height) {
   return width * height / 2;
}
```

```Java
//Javaでの関数定義
double triangleArea(int width, int height) {
   return width * height / 2;
}
```

## 引数（parameter）

外部とやりとりを行う特殊な変数のこと

e.g.)関数 triangleArea(5,4) が呼び出されたとき、width = 5, height = 4 として、関数内で width * height / 2 つまり 5 * 4 / 2 が計算され、関数の出力として 10 が返されます。

また、ほとんどの言語では、returnを使用することで、返り値を指定する。この時、関数が呼び出し元に対して返す値を==**戻り値**====  
（return value）==といいます。

さらに、Rubyにおいて、returnは存在するが日常的に使われるものではなく、最後に評価された式が戻り値として返されるという仕様

## クラスと名前空間

---

ソフトウェア開発は、複数の開発者によって開発されるため、異なる処理を持つ関数に同じ名前を付けてしまうことが予想されます。このような関数名の衝突を避けるために、名前空間が使われます。

e.g.)円、正三角形、正方形それぞれに対して、外周と面積を求める関数を名前衝突することなく定義する例

```Java
//Javaで名前空間を作る

// 円に関する関数
namespace Circle {
   double perimeter(int radius) {}
   double area (int radius) {}
}
 
// 正方形に関する関数
namespace Square {
   int perimeter(int side) {}
   int area (int side) {}
}
 
// 正三角形に関する関数
namespace EquilateralTriangle {
   int perimeter(int side) {}
   double area (int side) {}
}
```

名前空間内の関数にアクセスする際は、Circle.perimeter や Square.area のように、メンバ(.)演算子を使ってアクセスします。

### 名前空間としてのクラス

また、データと関数を持つ構造のことを**クラス**（class）と呼びます。この二つのデータと関数という要素はプログラミングにおいて重要なため、これらをまとめる場所となるクラスは非常に重要です。

データがクラスの中で定義される時、

それは**メンバ変数**（member variable）といい、

クラスによって囲まれた関数は

**メソッド**（method）といいます。

Rubyにおいては、ローカル変数と、クラスメソッドの関係に当たるかな?

クラスには、インスタンス化、継承、カプセル化、インターフェース等、より多くの機能があります

e.g.)クラスを名前空間として利用する場合の例

```JavaScript
// クラス構造を利用して、名前空間として使うことができます

class Circle {
   // クラス内ではデータはメンバ変数と呼ばれます
   // クラスを名前空間として使う場合は、static をつけます
   static PI = 3.14;
 
   // メソッドの定義
   static perimeter(radius){
      // 円周は 2πr
      // メンバ変数へのアクセスは、メンバ(.)演算子を使います
      return 2 * Circle.PI * radius;
   }
   
   static area(radius){
      // 円の面積は πr^2
      return Circle.PI * radius * radius;
   }
}

class Square {
   static perimeter(side) {
      // 正方形の周囲の長さは 4 × 1 辺
      return 4 * side;
   }
   
   static area(side) {
      // 正方形の面積は (1 辺)^2
      return side * side;
   }
}

class EquilateralTriangle {
   static perimeter(side){
      // 正三角形の周囲の長さは 3 × 1 辺
      return 3 * side;
   }
      
   static area(side){
      // 正三角形の面積は √3/4 × (1 辺)^2
      // √x は x^0.5 なので、べき乗(**)演算子を使って求めることができます
      return (3 ** 0.5 / 4) * side * side;
   }
}

// クラス（名前空間）の中にある関数にアクセスします
console.log(Circle.perimeter(4));
console.log(Circle.area(4));

console.log(Square.perimeter(4));
console.log(Square.area(4));
 
console.log(EquilateralTriangle.perimeter(4));
console.log(EquilateralTriangle.area(4));

// クラス（名前空間）の中にあるメンバ変数にアクセスします
console.log(Circle.PI);
```

- 関数の目的

- 関数の入力

- 関数の出力

を意識すること。

## find関数

---

文字列 S1 を入力として受け取り、文字列の中に S1 が含まれていた場合、最初に見つかった位置(index)を返します。含まれていない場合は、-1 を返します。

JavaScriptでいうとindexOfメソッド

```JavaScript
// recursion@info.com の @ のインデックスの位置を調べましょう
let email = "recursion@info.com";
// console.log で出力してください
console.log(email.indexOf("@"));

//マッチするものがない時は　 -1を返す　
```

RubyでいうString#index

```Ruby
"astrochemistry".index("str") # =>1
#マッチしない時は nilを返す
```

## replace関数

---

文字列 S1、文字列 S2 を入力として受け取り、文字列内の S1 を S2 で置換します。

JavaScriptの場合

replaceメソッド

```JavaScript
// 1 つ目だけを入れ替えます
console.log("Aaaaahhh the ocean".replace("a", "b"));

// /string/g で全ての文字列を入れ替えます
console.log("Aaaaahhh the ocean".replace(/a/g, "b"));
//正規表現をマッチするものを全て"g" 
もしくは
/regexpオブジェクトを作成?
console.log("Aaaaahhh the ocean".replace(new RegExp("a", "g"), "b");
```

Rubyの場合

String#sub String#gsub

```Ruby
# 1 つ目だけを入れ替えます
"Aaaaahhh the ocean".sub("a", "b")
# マッチしたものすべてを入れ換える
"Aaaaahhh the ocean".gsub("a", "b")
```
