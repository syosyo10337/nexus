# IPアドレスについて

127.0.0.1 —ループバックアドレス(loopback address)。つまり自分自身を表すIP(openfileのIPがこれになっているということは、他からの接続は受け付けていないことを示す。)

## プライベートIPアドレス

LAN内で使用されるアドレスのこと。

|   |   |
|---|---|
|クラス|アドレス範囲|
|クラスA|10.0.0.0 ～ 10.255.255.255|
|クラスB|172.16.0.0 ～ 172.31.255.255|
|クラスC|192.168.0.0 ～ 192.168.255.255|
