# ブランチのリネーム・リモートブランチの削除

# リモートブランチの削除

```Shell
git push --delete origin <branch_name>
```

# ローカルの削除

```Shell
git branch -d <localBranchName>
```

```Bash
git branch -m <古いブランチ名> <新しいブランチ名>
git branch -m <新しいブランチ名>
```
