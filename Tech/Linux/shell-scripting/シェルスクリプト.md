# シェルスクリプト

シェルスクリプトの基礎知識と基本的な使い方について説明します。

---

## シェルとは

シェルは、カーネルが提供する機能を操作する際に、OSと対話的に操作するためのインターフェースです。シェルはOSの特にカーネル部分を包み込んでいることからその名があり、対話機能を提供します。

### 主要なシェル

| シェル | 説明 |
|--------|------|
| `bash` | Bourne Again Shell（最も一般的） |
| `zsh` | Z Shell（macOSのデフォルト） |
| `sh` | Bourne Shell（POSIX互換） |
| `fish` | Friendly Interactive Shell |

---

## プログラムの主要4要素

シェルスクリプトは、以下の4つの要素で構成されます:

- **順次実行**: コマンドを上から順に実行
- **条件分岐**: 条件に応じて処理を分岐 → [条件分岐.md](条件分岐.md)
- **繰り返し**: 同じ処理を繰り返す → [繰り返し処理.md](繰り返し処理.md)
- **サブルーチン**: 処理をまとめて再利用可能にする（関数） → [関数.md](関数.md)
