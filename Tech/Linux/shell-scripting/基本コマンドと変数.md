# 基本コマンドと変数

> **関連**: [シェルスクリプト.md](シェルスクリプト.md)｜[シェル変数.md](シェル変数.md)｜[ベストプラクティス.md](ベストプラクティス.md)（クォート・命名規則）

echo、変数の基本、read、引用符の種類と使い分けをまとめます。

---

## echoコマンド

引数で与えた文字列を標準出力に出力します。

```bash
echo [オプション] 文字列
```

### 主要オプション

- `-n`: 改行を抑制（通常は自動で改行されます）
- `-e`: エスケープシーケンスを有効化

### 使用例

```bash
$ echo "Hello, World!"
Hello, World!

$ echo -n "改行なし"
改行なし$
```

## 変数の基本操作

> **詳細**: シェル変数と環境変数の詳細は[シェル変数.md](シェル変数.md)を参照してください。

### 変数の代入と参照

```bash
# 代入は「=」、参照は「$」を使用
$ abc=123
$ echo $abc
123
```

> **注意**: `=`の前後にスペースを入れないこと（コマンドとして解釈されてしまいます）

### 配列変数

Bashでは1次元の配列変数が使用できます。

```bash
# 配列への代入
$ abc[0]=123
$ abc[1]=456

# 配列の参照（波括弧で囲む）
$ echo ${abc[0]}
123
$ echo ${abc[1]}
456
```

## readコマンド

標準入力からデータを読み込み、変数に格納します。既存のデータは上書きされます。

```bash
read 変数名
```

> **注意**: 変数名の前に`$`は不要です（参照ではなく代入のため）

### 使用例

```bash
$ echo $abc
123

$ read abc
aaabbbccc

$ echo $abc
aaabbbccc
```

## 引用符（クォート）の種類と使い分け

### シングルクォート（`'`）

囲んだ範囲を**文字列として**そのまま認識します。

```bash
$ echo 'Value is $abc'
Value is $abc
```

### ダブルクォート（`"`）

囲んだ範囲の`$`付き変数を**変数として展開**します。

```bash
$ abc=123
$ echo "Value is $abc"
Value is 123
```

### バッククォート（`` ` ``）

囲まれた範囲を**コマンドとして実行**し、その結果を返します。

```bash
$ XYZ=`date`
$ echo "It is $XYZ now."
It is Thu Mar 20 06:08:14 JST 2017 now.

# 変数への代入を省略する書き方
$ echo "It is `date` now."
It is Thu Mar 20 06:08:14 JST 2017 now.
```

> **現代的な書き方**: バッククォートの代わりに`$(command)`の使用が推奨されます。
