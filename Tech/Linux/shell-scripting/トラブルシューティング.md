## よくあるエラー

### 1. "command not found"

```bash
# 原因: PATHが通っていない、または実行権限がない
# 解決: フルパスで実行するか、権限を確認

$ chmod +x script.sh
$ ./script.sh
```

### 2. "Permission denied"

```bash
# 原因: 実行権限がない
# 解決: 実行権限を付与

$ chmod u+x script.sh
```

### 3. "bad interpreter"

```bash
# 原因: シェバンのパスが間違っている
# 解決: 正しいパスに修正

#!/usr/bin/env bash  # 推奨
```

### 4. "syntax error near unexpected token"

```bash
# 原因: 構文エラー（スペース、クォート、括弧など）
# 解決: ShellCheckで確認

$ shellcheck script.sh
```
