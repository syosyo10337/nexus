# スクリプトの引数処理とエスケープ

> **関連**: [シェルスクリプト.md](シェルスクリプト.md)｜[特殊変数.md](特殊変数.md)｜[基本コマンドと変数.md](基本コマンドと変数.md)（クォート）

引数（位置パラメータ）・shift、およびエスケープ文字・エスケープシーケンスをまとめます。

---

## 位置パラメータ（引数）

スクリプト実行時に渡された引数を参照できます。

### 特殊変数

| 変数 | 説明 |
|------|------|
| `$0` | 実行コマンド名（スクリプト名） |
| `$1`, `$2`, `$3`... | 1番目、2番目、3番目...の引数 |
| `$#` | 引数の個数 |
| `$@` | すべての引数（個別に展開） |
| `$*` | すべての引数（1つの文字列として展開） |

### 使用例

```bash
#!/bin/bash
echo "$1:" $1
echo "$2:" $2
echo "$3:" $3
echo "$0:" $0
echo "$#:" $#
```

実行結果:

```bash
$ ./args.sh aaa bbb ccc
$1: aaa
$2: bbb
$3: ccc
$0: ./args.sh
$#: 3
```

## shiftコマンド

引数の順序を左にずらします（`$1`を削除し、`$2`が`$1`になる）。

### 使用例

```bash
#!/bin/bash
echo "$1:" $1
echo "$2:" $2
echo "$3:" $3
shift
echo "$1:" $1
echo "$2:" $2
```

実行結果:

```bash
$ ./argsshift.sh aaa bbb ccc
$1: aaa
$2: bbb
$3: ccc
$1: bbb  # $2だったものが$1に
$2: ccc  # $3だったものが$2に
```

### 動作の説明

- **shift前**: `$1=aaa`, `$2=bbb`, `$3=ccc`
- **shift後**: `$1=bbb`, `$2=ccc`（`aaa`は削除される）

> **ユースケース**: 多数の引数を順次処理する際に便利です。

# エスケープ文字とエスケープシーケンス

## エスケープ文字（`\`）

バックスラッシュ（`\`）は、直後の1文字の扱いを変更します。

### 行の継続

行末にバックスラッシュを付けることで、コマンドを複数行に分割できます。

```bash
$ echo "I am a cat. As yet I have no name."
I am a cat. As yet I have no name.

$ echo "I am a cat. \
> As yet I have no name."
I am a cat. As yet I have no name.
```

> **利点**: 長いコマンドの視認性を向上できます。

### 特殊文字のエスケープ

```bash
# ダブルクォートを出力する方法1: シングルクォートで囲む
$ echo 'Value of ABC is "$ABC".'
Value of ABC is "$ABC".

# ダブルクォートを出力する方法2: バックスラッシュでエスケープ
$ echo "Value of ABC is \"$ABC\"."
Value of ABC is "123".

# $記号をそのまま表示
$ echo "Value of ABC is \$ABC."
Value of ABC is $ABC.
```

## エスケープシーケンス

`echo -e`オプションを使用すると、エスケープシーケンスが有効になります。

### 主要なエスケープシーケンス

| シーケンス | 説明 |
|-----------|------|
| `\n` | 改行 |
| `\t` | タブ |
| `\r` | キャリッジリターン |
| `\\` | バックスラッシュ |
| `\0nnn` | 8進数で文字を指定（例: `\041`は`!`） |

### 使用例

```bash
$ echo -e "I am a cat. \nAs yet I have no name\041"
I am a cat.
As yet I have no name!
```
