# スクリプトの作成と実行

> **関連**: [シェルスクリプト.md](シェルスクリプト.md)（概要）｜[ベストプラクティス.md](ベストプラクティス.md)（厳格モード・冒頭の書き方）

シェルスクリプトの作成方法、シェバン、実行権限、実行のしかたをまとめます。

---

## 基本的な作成方法

シェルスクリプトはテキストファイルとして作成します（`vi`や`vim`などのエディタを使用）。

### スクリプトの基本構造

```bash
#!/bin/bash
# 1行目: シェバン（shebang）でシェルを指定

# コメント: #で始まる行はコメント

# 2行目以降: 実行するコマンドを記述
echo "Hello, World!"
date
ls -l
```

### シェバン（Shebang）

| シェバン              | 説明                       |
| --------------------- | -------------------------- |
| `#!/bin/bash`         | Bash で実行                |
| `#!/bin/sh`           | POSIX 互換シェルで実行     |
| `#!/usr/bin/env bash` | 環境の bash を使用（推奨） |
| `#!/bin/zsh`          | Zsh で実行                 |

> **推奨**: `#!/usr/bin/env bash`を使用すると、システムに依存せず bash を見つけられます。

## 実行権限の付与

```bash
# 所有者に実行権限を付与（推奨）
$ chmod u+x script.sh

# 全員に実行権限を付与
$ chmod +x script.sh

# 数値指定
$ chmod 755 script.sh  # rwxr-xr-x

# 権限を確認
$ ls -l script.sh
-rwxr-xr-x 1 user group 100 Jan 4 10:00 script.sh
```

> **注意**: スクリプトを実行するには、実行権限（`x`）が必要です。

## スクリプトの実行方法

### 方法 1: パス指定で実行

```bash
# カレントディレクトリのスクリプトを実行
$ ./script.sh

# 絶対パスで実行
$ /path/to/script.sh

# PATHが通っているディレクトリなら
$ script.sh
```

### 方法 2: シェルを明示的に指定

```bash
# Bashで実行
$ bash script.sh

# shで実行
$ sh script.sh

# 実行権限不要
```

### 方法 3: source コマンド

```bash
# 現在のシェル環境で実行
$ source script.sh

# または短縮形
$ . script.sh
```

> **source の特徴**: 変数や関数が現在のシェルに残ります。詳細は[source コマンド.md](sourceコマンド.md)を参照。
