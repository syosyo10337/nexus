# プロセス

プロセスとは、**実行中のプログラムのこと**。1つのプロセスに1つのメモリ領域が割り当てられる。

スレッドとは、プロセス内で命令を逐次実行する部分であり、CPUのコアを利用する単位。連続して実行される一本の命令の並び、処理の流れ
