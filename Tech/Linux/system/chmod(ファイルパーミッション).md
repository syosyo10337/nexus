# chmod(ファイルパーミッション)

```JavaScript
3つのカテゴリ
rwx rwx rwx
│   │   └─ Others (その他): システム上の他の全ユーザー
│   └───── Group (グループ): ファイルが属するグループのメンバー
└───────── Owner (所有者): ファイルの所有者（通常は作成者）
```

# 典型的な設定例

|   |   |   |
|---|---|---|
|用途|パーミッション|理由|
|SSH秘密鍵|400 or 600||
|設定ファイル|600 644||
|web コンテンツ|644(file) 755(dir)|webサーバが読めるように|
