# wcコマンド

## 概要

✅ **wc = Word Count (ワードカウント)**

テキストの行数、単語数、文字数（バイト数）を数えるコマンドです。
ファイル解析やスクリプトでのカウント処理に広く使用される基本コマンドです。

## 基本構文

```bash
wc [オプション] [ファイル名...]
```

## 主なオプション

| オプション | 説明 |
|-----------|------|
| `-l` | 行数のみを表示 (lines) |
| `-w` | 単語数のみを表示 (words) |
| `-c` | バイト数のみを表示 (bytes) |
| `-m` | 文字数のみを表示 (characters) |
| `-L` | 最長行の長さを表示 |

## 基本的な使い方

```bash
# すべての情報を表示（行数、単語数、バイト数）
wc file.txt
# 出力:      34     120    1500 file.txt

# 行数のみを表示
wc -l file.txt
# 出力:      34 file.txt

## 頻出パターン

### パターン1: ファイルを直接指定

```bash
wc -l file.txt
# 出力:      34 file.txt
#           ↑  ↑
#           行数 ファイル名
```

### パターン2: リダイレクトで標準入力から読み込む ⭐重要
特定のファイルから、行数だけを抽出する際に使われる

```bash
wc -l < file.txt
# 出力:      34
#           ↑
#           行数のみ（ファイル名なし）
```

**`<` の意味:**
- `<` = 入力リダイレクト（input redirection）
- ファイルの内容を標準入力としてコマンドに渡す
- `wc`はファイル名を知らないため、数値だけを出力
- **注意:** 左側にスペースが入る（数値の前に空白）
