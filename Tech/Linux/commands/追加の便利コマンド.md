# 追加の便利コマンド

## ripgrep (rg)

超高速なgrepの代替ツール。 ファイル内のテキスト検索に使います。

### 基本的な使い方

```bash
# カレントディレクトリ以下を再帰検索
rg "検索パターン"

# 特定のファイルタイプだけ検索
rg "function" --type js
rg "def " --type py

# 大文字小文字を無視
rg -i "hello"

# ファイル名だけ表示
rg -l "TODO"

# 前後の行も表示（コンテキスト）
rg -C 3 "error" # 前後3行
rg -B 2 "error" # 前2行
rg -A 2 "error" # 後2行

# 特定ディレクトリ/ファイルを除外
rg "pattern" --glob '!node_modules'
```

grep -r より圧倒的に速いのがポイント。.gitignore を自動で尊重してくれるのも便利です。

## zoxide (z)

cd の賢い代替ツール。 過去に移動したディレクトリを覚えていて、部分一致でジャンプできます。

### 基本的な使い方

```bash
# 一度でも行ったことがあるディレクトリにジャンプ
z wezterm # ~/.config/wezterm にジャンプ（部分一致）
z dotfiles # ~/dotfiles にジャンプ

# 複数キーワードで絞り込み
z config nvim # ~/.config/nvim にジャンプ

# インタラクティブに選択（fzf連携）
zi

# 通常の cd のようにも使える
z ~/Projects/myapp
```

### セットアップ

```bash
# インストール（macOS）
brew install zoxide

# シェルに初期化を追加（.zshrc など）
eval "$(zoxide init zsh)"
```

使えば使うほどディレクトリの「頻度＋最近度」を学習して、少ないキーストロークで目的のディレクトリに飛べるようになります。
