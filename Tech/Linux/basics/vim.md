# vimエディタ

vimエディタ--ページャであり、エディタでもあります。vi でファイルを開いた場合はコマンドモード（viewモード）といって、ページングによる行やページ単位の移動や、行の削除、コピー、カット、ペーストなどの編集操作を行なうことができるモードと文字入力を受け付けるインサートモードがあります。

## 基本操作

### vim(viでも可) [ファイル名]
vimエディタを指定ファイルに対して起動する。もしくは新規作成できる。

### エディタの終了操作
- ファイルの閉じ方: `:q` と入力します。(quit)
- ファイルを保存する: `:w` (spaceの後にファイル名を指定して名づけることも可能)
- ファイルを保存して終了する: `:wq`
- ファイルを保存せずに強制終了: `:q!`

## インサートモードとコマンドモードの切り替え

### インサートモードの操作
- `ESC` -- インサートもモードを抜ける。
- `i` -- カーソルがある位置に挿入
- `a` -- カーソルがある文字の後に挿入
- `o` -- カーソルの下に新しい行を作成してインサートモードに入る
- `O` (shift + o) -- カーソルの上に新しい行を作成してインサートモードに入る
- `I` (shift + i) -- 行頭に入力するとき
- `A` (shift + a) -- 行の最後にテキストを入力するとき

## コマンドモード(ノーマルモード)の操作

### カーソルの移動
（centOSはカーソルキー使えるんだけど、、）
- ←↓↑→ --- `h`(左) `j`(下) `k`(上) `l`(右)
- `0` --- カレント行の行頭へ移動
- `$` (shift + 4) --- カレント行の行末へ移動
- `w` -- 次の単語の先頭へ移動
- `b` -- 前の単語の先頭へ移動
- `e` -- 次の単語の末尾へ移動

### ページ単位での移動
- `ctrl + f` --- 次のページ
- `ctrl + b` -- 前のページ

### 行単位の移動
- `:行数` ex) `:10` -- １０行目に移動
- `gg` -- 文書頭、１行目
- `G` (shift + g) -- 文書末　最終行

## さまざまな編集操作

### カット、アンド、ペースト
- `x` -- 1 文字削除
- `dd` -- 1 行削除
- `yy` -- 1 行コピー
- `nyy` -- n 行コピー
- `p` -- カーソルの文字の次または次の行にペースト
- `P` (shift + p) -- カーソルの文字の前または前の行にペースト
- `u` -- 直近の操作を一回取り消し（アンドゥ）
- `U` -- 現在の変更行の全ての編集を取り消す

#### カット
- `d$` -- カーソルから行末までカット
- `d0` -- カーソルから行頭までカット

### 検索
- `/` -- 検索文字列文字列の検索 ハイライトを消すには`:nohlsearch` (No High Light search)
- `n` -- 下方向へ再検索
- `N` -- 上方向へ再検索

### 置換
`:対象の行 s/検索文字列/置換文字列/オプション` --- 文字列を置換する

- `:ns/old/new` -- n 行目の最初のold をnew に置換して終了
- `:ns/old/new/g` -- n 行目の全てのold をnew に置換して終了
- `:%s/old/new/g` -- ファイル全体の検索語句を置換する
- `:%s/old/new/gc` -- 置換の度に確認を求める

対象の行を省略すると現在カーソルがある行だけが置換対象になる。`g`のオプションを指定すると、１行内の全てを置換対象に取ります。`%`はファイル全体を指定します。
