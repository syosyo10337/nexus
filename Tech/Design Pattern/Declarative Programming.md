# Declarative Programming(宣言的プログラミング)

そもそも宣言的プログラミングとは何か。

- 何を達成したいかを記述する(What)
- 最終的な結果の状態の状態を記述して、実現方法は任せる
- コードはこうみえてほしいという形で書かれる。

具体的な、宣言的アプローチを採用している web 関連技術としては、

- kubernetes
- react

## 対比項: 命令的プログラミング(Imperative Programming)

- "どのように(How)" 達成するかを記述する
- ステップバイステップで具体的な手順を指示する
- コードは「これをして、次にこれをして...」という形で書かれる
