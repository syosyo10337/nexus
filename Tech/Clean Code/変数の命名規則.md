# 変数の命名規則

- CamelCase
    
    - UpperCamelCase(PascalCase)  
        単語の先頭を大文字にする(e.g. C#)
    
    - lowerCamelCase  
        単語の先頭を大文字するが、変数名の最初の文字だけ小文字  
        (e.g. Java, JavaScript)
    

- snake_case  
    単語をアンダーバーで繋ぐ(e.g. Python, Ruby)

- kebab-case  
    単語をハイフンで繋ぐ(e.g. HTML CSS)
