Service Objectをインスタンス化しても、単にcallメソッドを実行する以外にあまり使いみちがありません。callメソッドを実行するのであれば、次のように抽象化してService Objectの呼び出しをより簡潔にすることを検討しましょう。

```Ruby
module Service
	extend ActiveSupport::Concern
	class_methods do
		def call(*args)
			new(*args).call
		end
	end
end
```
