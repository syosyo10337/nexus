# 相関係数と特徴量

**特徴量と正解データの相関係数が大きいほど、つまり、これらx、yの値の相関が強いほど、特徴量が与える影響が大きく、モデルの性能が高くなる**

## `pd.DataFrame.corr()`

correlation: 相関のこと。各列同士の相関係数を調べることができる。

戻り値は各列同士の相関係数を値として持つ、データフレーム　

```Python
e.g.)

pd.corr()
```

[![](ML/content/Attachments/Screenshot_2023-02-15_at_18.25.52.png)](%E7%9B%B8%E9%96%A2%E4%BF%82%E6%95%B0%E3%81%A8%E7%89%B9%E5%BE%B4%E9%87%8F/Screenshot_2023-02-15_at_18.25.52.png)

corr()関数を呼び出すと、上のような図が出力される。これを**==相関行列==**と呼ばれる

[map処理](../pandas\(%E8%B6%85\)%E5%85%A5%E9%96%80%20f3b7db9279004a0cbcb4c8f282984e5e.html)を使って、相関係数を相関の強い順に並び替えを行う

```Python
e.g.)
train_corr = pd.corr()['Y']
train_corr.map(abs).sort_values()
```

## `se.sort_values(ascending = True)`

シリーズの数値を並び替える。defaultだと昇順だが、`ascending`引数にFalseを渡して呼び出すと降順でも、表示ができる。
