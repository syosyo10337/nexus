# グループ集計

---

[`pd.pivot_table(df,   index= ‘col’,   columns=’col’,   values=’集計するcol’,   aggfunc=’集計関数名’   )`](#64d79858-817d-41d0-ab97-c39fd5c7951f)

欠損値を補完する際に、ある属性(列)によって、特定の属性の平均値や中央値にブレが出る時がある。

データ全体に対して、平均寿命を出した場合と男女別に平均寿命を出した場合の間に、女性の平均寿命のほうが男性よりも長いことが、一例として挙げられる。

このように、データ全体に対しての平均値では、細かな傾向を無視しておおざっぱな平均値/中央値というような代表値を取ることになる。

機械学習のモデルに学習させる場合に、欠損値を全体のデータによる穴埋めで補完すると、学習の際に精度のブレにつながる。

これを防止するためには、クロス集計などによって、欠損値をより傾向に沿った代表値で埋めていく。

## `df.groupby(’col’).<集計関数>()`

colに、グループ分けをしたい基準となる列を取り、その列の区別ごとに、それぞれの値を集計する。

```Python
e.g.)

# Survivedつまり、生存したか否かによって['Age']カラムの平均値を算出
df2.groupby('Survived').mean(numeric_only=True)['Age']
# 客室のクラスによって['Age']カラムの平均値を算出
df2.groupby('Pclass').mean(numeric_only=True)['Age']
```

## `pd.pivot_table(df,   index= ‘col’,   columns=’col’,   values=’集計するcol’,   aggfunc=’集計関数名’   )`

index,columnsに基準としたい2つの列を選択し、valuesに集計結果を表示したい列を指定。デフォルトではmean()平均値を算出され、aggfuncの引数の値を指定することで、別の代表値を値として表示することができる。

```Python
e.g.)

pd.pivot_table(df2, index= 'Survived',columns = 'Pclass', values = 'Age', aggfunc = max)
```
