# 交差作用特徴量

---

特徴量同士を掛け合わせた値を、新しい特徴量とすることを交互作用特徴量といいます。

[Series算術演算](../pandas\(%E8%B6%85\)%E5%85%A5%E9%96%80%20f3b7db9279004a0cbcb4c8f282984e5e.html)を仕様を利用して付け加える

```Python
x['RM * LSTAT'] = x['RM'] * x['LSTAT']
x.head(2)
```

意味合いとしては、’部屋数が多くて、かつ都会である’とか、’部屋数が多いが田舎である’というような特徴量同士の作用についても勘定に入れている。

部屋数についての変数`a`がとても大きいとき,　都会であるという変数`b`が大きいということがほとんどあり得ない。などというな関係性についても予想性能に反映させるための手法
