## `DataFrame.isnull()`

このメソッドはNaNやNoneなどのN/Aとなりうる値に対してTrue、それ以外はFalseとした

値の**DataFrameを返す**メソッド。

## 欠損値の有無の確認

isnull()メソッドで生成したDataFrameに対して、anyメソッドをチェーンすることで実装する。

### `DataFrame.isnull().any(axis = 0)`

e.g)

```Python
df.isnull().any(axis = 0)

#->
がく片長さ     True
がく片幅      True
花弁長さ      True
花弁幅       True
種類       False
dtype: bool
```

- `any(axis = 0)`
    
    anyメソッドは、指定した列もしくは行に対して、Trueの値があるのかを走査し、一つでもTrueがあった場合にはTrueを返す。
    
    - ==axis = 0 の時、列を基準に列ごとに走査を行う( reduce index)==
    
    - ==axis = 1 の時、行を基準に行ごとに走査を行う( reduce columns)==
    

## 欠損値の個数を集計する

isnull()メソッドで生成したDataFrameに対して、sum()メソッドを使うことで、実装する。

### `DataFrame.isnull().sum()`

```Python
# sum()メソッドを使う。
df.isnull().sum()

# ->
がく片長さ    2
がく片幅     1
花弁長さ     2
花弁幅      2
種類       0
dtype: int64
```

- `.sum()`
    
    各列に対して、列ごとのvalue  
    True-> 1/ False ->0として各列に対して足しあげることができる。  
    ==つまり、sumの結果が0の時、全てFalseだったことになる==
    

## `DataFrame.dropna(   axis=0,   how='any',   inplace=False   )`

**欠損値のある列もしくは行を削除する.**

==戻り値として、欠損値を削除したDataFrameを返す。==

- axis
    
    - 0の時、欠損値のあるindex/row/行を削除する
    
    - 1の時、 columnsを削除
    

- how =
    
    - ‘any’ —一つでも、該当する値があれば、その行/列を削除する
    
    - ‘all’ —もし、全ての値がNAであれば、その行/列を削除する
    

- inplace
    
    破壊的変更(つまり、元のdfを変更するか)
    
    Trueにすることで元のdfを変更することもできる。
    

## `DataFrame[<columns>].fillna(value)`

欠損値のある列に対して、valueでとった引数の値を代入したDataFrameを返す。

```Python
e.g.)

df['花弁長さ'] = df['花弁長さ'].fillna(0)
df.tail(3)
```

# 欠損値への対処

一般的には、欠損値の穴埋めが学習結果に影響を与えないようにするため、

- 数値の列には、その列の
    
    - 平均値
    
    - 中央値
    

のどちらかで埋める。

- 文字列の列に対しては、最頻値(mode)を入れます。

```Python
e.g.)
# 欠損値うめるための平均値を列ごとに算出(series)
col_mean = df.mean()

# 欠損値を平均値によって穴埋め
df2 = df.fillna(col_mean)

# 欠損値がないことを確認
df2.isnull().any(axis = 0)
```
