# データフレームの結合

## `pd.concat(objs, axis = 0)`

objs、つまりダミー変数化を施したデータフレームと元々の特徴量を取り出したデータフレームなどを与えて、結合させる

- axis = 1の時, dataFrameを縦軸基準として( along columns)

- axis = 0の時, dataFrameを横軸を基準として結合( along index)
