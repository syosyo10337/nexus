# データの前処理

前処理には次のようなものがあります。

- データの結合

- 欠損値の処理

- 外れ値の処理

- データの標準化

- 特徴量エンジニアリング

### 特徴量を取り除く

欠損データ（欠損値、欠測値、missing data）に対処する最も簡単な方法の1つは該当する特徴量(列）もしくは、サンプル（行）をデータセットから完全に削除することです。

e.g.)pandasでは、

**dropna**メソッドを使用することで、欠損値を含んでいる行を簡単に削除することが出来ます。

```Python
# 欠損値を含む行を削除
df.dropna()
# 同様に、axis引数を1に設定することで、NaNを含んでいる行が1つでもある列を削除することが出来ます。

# 欠損値を含む列を削除
df.dropna(axis=1)

# 全ての列がNaNの行のみを削除
df.dropna(how="all")

# 非NaN値が5つ未満の行のみを削除
df.dropna(thresh=5)

# 特定の列（ここでは"A")にNaNが含まれている行のみを削除
df.dropna(subset=["A"])
```

### 補間法

**補間法(interploation technique)は、欠損値への対処に最も使用される手法です。**

サンプルの行を削除したり、特徴量の列を削除しすぎたりすると、貴重なデータをあまりにも多く失ってしまうことになります。

このような場合に、補間法を用いることで、データセットの他のトレーニングサンプルから欠損値を推測することができます。

まずは最も一般的な**平均補間法(mean imputation)**の説明をします。

**平均補間法は、欠損値を特徴量の列全体の平均値と置き換えます。**

`sckit-learn`の`SimpleImputer`クラスを使用します。
