# 特徴量重要度

---

決定木における分岐条件においては、ルートノードに近い条件ほど、正解データの分類に影響を与えると解釈される。

この、分類に影響を与える度合いを**特徴量重要度**といいます。一般にはこの数値指標を用いてモデルの評価を行う。

scikit-learnにおいてもこの特徴量重要度を取得することができる。

## `model.feature_importances_`

numpyのarray型のオブジェクトを返す。

これは、特徴量にとったカラムの順に並んでいる。

```Bash
model.feature_importances_

# ->
array([0.09019032, 0.28797249, 0.0630141 , 0.0117882 , 0.17890329,
       0.3681316 ])
```
