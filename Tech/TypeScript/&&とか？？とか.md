# &&とか？？とか

✅ **JSX内での** `**&&**` **による条件付きレンダリング:Conditional Rendering with Logical AND** または **Short-circuit evaluation** と呼ばれます。

```TypeScript


tsx
// 条件付きレンダリング
{isLoggedIn && <WelcomeMessage />}
{items.length > 0 && <ItemList items={items} />}
```

✅ `**??**` **は Nullish coalescing operator（null合体演算子）:**  
nullまたはundefinedの場合のみデフォルト値を提供します。

```TypeScript


tsx
// null/undefinedの場合のみデフォルト値
const name = user.name ?? 'Guest';
{title ?? 'Default Title'}
```

**重要な違い:**

```TypeScript


tsx
// && は falsy値（0, '', false, null, undefined）で止まる
{count && <div>Count: {count}</div>}// count=0の時、0が表示される// ?? は null/undefinedのみをチェック
{count ?? 'No count'}// count=0の時、0が表示される（'No count'ではない）
```
