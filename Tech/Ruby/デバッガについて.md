# (draft)デバッガについて

`debug` gemやbyebug pryなど様々存在します。

Railsに限っては

```Ruby
Rails.logger.debug()
```

を使うことで、ログに出力することができる(putsのログ出力版)

- `binding.irb`  
    これは、default gem化されたので、とりあえずステップ実行するデバッグをしたい時には、これを使うと良い。
