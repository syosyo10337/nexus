#  CSV標準仕様 (RFC 4180)

### **正式なMIME Type**

-  **`text/csv`** - RFC 4180で定義された正式なMIME type (2005年策定)

## **基本ルール**

1. **改行** - CRLF (`\r\n`) を使用
2. **区切り文字** - カンマ (`,`) を使用
3. **フィールドのクォート**:
    - カンマ、改行、ダブルクォート(`"`)を含むフィールドは`"`で囲む
    - フィールド内の`"`は`""`としてエスケープ
4. **ヘッダー行** - オプション
5. **文字エンコーディング** - UTF-8を推奨(仕様では規定なし)

### **BOM (Byte Order Mark)について**

- **UTF-8 BOM**: `0xEF 0xBB 0xBF` (3バイト)
- ⚠️ **Excel互換性**: ExcelでUTF-8 CSVを正しく開くにはBOMが必要
- ✅ **Webアプリ**: BOMなしのUTF-8が標準
- ⚠️ **注意**: BOMがあると一部のパーサーで最初のカラム名にBOMが含まれる問題が発生
