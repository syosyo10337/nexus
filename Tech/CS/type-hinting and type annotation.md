# type-hinting and type annotation

The main difference between type hinting and type annotation is that type hinting is optional and is used for function parameters and return values, while type annotations are required for variables and function definitions in some programming languages, such as Python.

In summary, type hinting is used to provide additional information to code editors and linters to catch potential errors, while type annotations are used to explicitly declare the data type of variables and function definitions.
