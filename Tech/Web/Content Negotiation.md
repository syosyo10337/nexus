# Content Negotiation

HTTPにおいて、、同じ URI におけるさまざまな[表現](https://developer.mozilla.org/ja/docs/Glossary/Representation_header)  
のリソースを提供するために使用する仕組みであり、ユーザーエージェントはどのリソースがユーザーにもっとも適しているか (例えば文書の言語はどれか、画像形式はどれか、コンテンツエンコード方式はどれか) を指定することができます。
