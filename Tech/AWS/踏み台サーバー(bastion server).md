# 踏み台サーバー(**bastion server**)

---

ネットワーク内のリソースに接続して、設定などを行う際に、リソースへの接続を制限したい(ex:管理者のみが可能など)。すべてのリソースに対して個別に設定する場合だと、効率も悪く、確実性にも欠けるため。

リソースに接続するための入口としてのサーバーを用意して、このサーバー経由でのみ各リソースへの接続を可能にする方式が好まれる。

この時に用いられるサーバーを**踏み台サーバー**と呼ぶ(リソースへの入り口が唯一の役割となるため、サーバーとしてのスペックは高い必要はない。）
