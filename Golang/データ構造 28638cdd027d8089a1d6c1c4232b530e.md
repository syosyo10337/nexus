# Slice

- サイズが動的

- 参照型（内部的には配列への参照を持つ）

- `append`で要素を追加できる

- Goで最もよく使われる ✅

```Go
// Rubyのarrayに最も近い、動的なサイズ
s := []int{1, 2, 3}           // slice literal
s = append(s, 4)              // 要素を追加できる
fmt.Println(len(s))           // 4
```

## Array（配列）- 限定的な使用

go

```Go
// 固定サイズ、サイズが型の一部
var a [3]int = [3]int{1, 2, 3}  // 長さ3の配列// a = append(a, 4)  // エラー！配列にはappendできない
```

**特徴：**

- サイズが固定で、型の一部（`[3]int`と`[4]int`は別の型）

- 値型（コピーが発生する）

- サイズ変更不可

- 低レベルな処理や特定の最適化が必要な場合のみ使用 ⚠️
