# 繰り返し処理

> **関連ドキュメント**: シェルスクリプトの基礎については[シェルスクリプト.md](シェルスクリプト.md)を参照してください。

繰り返し処理（ループ）を実装する方法について説明します。

---

## for文

リストの各要素に対して処理を繰り返します。

### 基本構文

```bash
for 変数 in 値のリスト
do
    処理
done
```

### 使用例

#### 基本的な使い方

```bash
# 文字列のリスト
$ for i in a b c d
> do
> echo $i
> done
a
b
c
d
```

#### コマンドの実行結果を使用

```bash
# バッククォート（非推奨）
for i in `ls`
do
    echo "File: $i"
done

# $(command) 形式（推奨）
for i in $(ls)
do
    echo "File: $i"
done

# より安全な方法（ファイル名にスペースが含まれる場合）
for i in *
do
    echo "File: $i"
done
```

#### 数値の範囲指定

```bash
# Bash拡張: {開始..終了}
for i in {1..5}
do
    echo "Number: $i"
done

# 増分指定
for i in {0..10..2}
do
    echo "Even number: $i"
done

# C言語スタイル（Bash拡張）
for ((i=1; i<=5; i++))
do
    echo "Count: $i"
done
```

#### 配列のループ

```bash
#!/bin/bash

fruits=("apple" "banana" "orange")

# 配列の要素をループ
for fruit in "${fruits[@]}"
do
    echo "Fruit: $fruit"
done

# インデックス付きループ
for i in "${!fruits[@]}"
do
    echo "Index $i: ${fruits[$i]}"
done
```

#### ファイル操作の例

```bash
#!/bin/bash

# カレントディレクトリの.txtファイルを処理
for file in *.txt
do
    if [ -f "$file" ]; then
        echo "Processing: $file"
        # ファイルの行数を表示
        wc -l "$file"
    fi
done
```

## while文

条件が真の間、処理を繰り返します（条件が偽になったら終了）。

### 基本構文

```bash
while [ 条件 ]
do
    処理
done
```

### 使用例

#### カウンターを使った繰り返し

```bash
#!/bin/bash

count=1
while [ $count -le 5 ]
do
    echo "Count: $count"
    count=$((count + 1))
done
```

#### 無限ループ

```bash
#!/bin/bash

# 方法1: trueコマンド
while true
do
    echo "Press Ctrl+C to stop"
    sleep 1
done

# 方法2: :コマンド（より効率的）
while :
do
    echo "Running..."
    sleep 1
done
```

#### ファイルを1行ずつ読み込む

```bash
#!/bin/bash

# 方法1: リダイレクト
while read line
do
    echo "Line: $line"
done < input.txt

# 方法2: パイプ（サブシェルで実行される点に注意）
cat input.txt | while read line
do
    echo "Line: $line"
done

# 方法3: IFSを使った区切り文字指定
while IFS=',' read col1 col2 col3
do
    echo "Column 1: $col1, Column 2: $col2, Column 3: $col3"
done < data.csv
```

#### ユーザー入力の処理

```bash
#!/bin/bash

while true
do
    read -p "コマンドを入力してください (quit で終了): " cmd
    
    if [ "$cmd" = "quit" ]; then
        break
    fi
    
    echo "実行: $cmd"
    eval "$cmd"
done
```

## until文

条件が偽の間、処理を繰り返します（条件が真になったら終了）。

### 基本構文

```bash
until [ 条件 ]
do
    処理
done
```

> **whileとの違い**: `while`は条件が真の間継続、`until`は条件が真になるまで継続

### 使用例

#### 基本的な使い方

```bash
#!/bin/bash

count=1
until [ $count -gt 5 ]
do
    echo "Count: $count"
    count=$((count + 1))
done
```

#### ファイルの存在を待つ

```bash
#!/bin/bash

file="/tmp/ready.flag"

echo "Waiting for $file..."
until [ -f "$file" ]
do
    echo "Still waiting..."
    sleep 2
done

echo "File found!"
```

#### サービスの起動を待つ

```bash
#!/bin/bash

port=8080

echo "Waiting for service on port $port..."
until nc -z localhost $port
do
    echo "Service not ready yet..."
    sleep 1
done

echo "Service is ready!"
```

## select文

ユーザーに選択肢を提示し、数値入力を促します。メニュー形式のインターフェースを簡単に作成できます。

### 基本構文

```bash
select 変数 in リスト
do
    処理
done
```

### 使用例

#### 基本的なメニュー

```bash
#!/bin/bash

select fruit in "apple" "banana" "orange" "quit"
do
    case $fruit in
        apple|banana|orange)
            echo "You selected $fruit"
            ;;
        quit)
            echo "Goodbye!"
            break
            ;;
        *)
            echo "Invalid selection"
            ;;
    esac
done
```

実行結果:

```
1) apple
2) banana
3) orange
4) quit
#? 1
You selected apple
#? 4
Goodbye!
```

#### PS3プロンプトのカスタマイズ

```bash
#!/bin/bash

PS3="果物を選んでください (番号入力): "

select fruit in "りんご" "バナナ" "オレンジ" "終了"
do
    case $fruit in
        りんご|バナナ|オレンジ)
            echo "${fruit}を選びました"
            ;;
        終了)
            echo "プログラムを終了します"
            break
            ;;
        *)
            echo "無効な選択です"
            ;;
    esac
done
```

#### 実用的なメニューシステム

```bash
#!/bin/bash

PS3="操作を選択してください: "

options=("ファイル一覧" "ディスク使用量" "メモリ使用量" "終了")

select opt in "${options[@]}"
do
    case $REPLY in
        1)
            echo "=== ファイル一覧 ==="
            ls -lh
            ;;
        2)
            echo "=== ディスク使用量 ==="
            df -h
            ;;
        3)
            echo "=== メモリ使用量 ==="
            free -h
            ;;
        4)
            echo "終了します"
            break
            ;;
        *)
            echo "無効な選択です: $REPLY"
            ;;
    esac
    echo ""
done
```

> **終了方法**: `Ctrl + C`で強制終了、または`break`で終了

## 繰り返しの制御

### break

ループを即座に終了します。

```bash
#!/bin/bash

for i in {1..10}
do
    if [ $i -eq 5 ]; then
        echo "Breaking at $i"
        break
    fi
    echo "Number: $i"
done
echo "Loop ended"
```

出力:

```
Number: 1
Number: 2
Number: 3
Number: 4
Breaking at 5
Loop ended
```

### continue

現在の反復をスキップし、次の反復に進みます。

```bash
#!/bin/bash

for i in {1..5}
do
    if [ $i -eq 3 ]; then
        echo "Skipping $i"
        continue
    fi
    echo "Number: $i"
done
```

出力:

```
Number: 1
Number: 2
Skipping 3
Number: 4
Number: 5
```

### ネストしたループでの使用

```bash
#!/bin/bash

# 外側のループを終了
for i in {1..3}
do
    echo "Outer loop: $i"
    for j in {1..3}
    do
        echo "  Inner loop: $j"
        if [ $j -eq 2 ]; then
            break  # 内側のループのみ終了
        fi
    done
done
```

### 実用例: ユーザー入力の検証

```bash
#!/bin/bash

while true
do
    echo "Continue? (y/n)"
    read input
    
    case $input in
        n|N)
            echo "Exiting..."
            break
            ;;
        y|Y)
            echo "Continuing..."
            continue
            ;;
        *)
            echo "Please input y or n."
            ;;
    esac
done
```

## 実践的なテクニック

### ループカウンターの管理

```bash
#!/bin/bash

# 方法1: 算術展開
count=0
for file in *.txt
do
    count=$((count + 1))
    echo "$count: $file"
done

# 方法2: letコマンド
count=0
for file in *.txt
do
    let count++
    echo "$count: $file"
done
```

### 並列処理

```bash
#!/bin/bash

# バックグラウンドで並列実行
for i in {1..5}
do
    (
        echo "Task $i started"
        sleep 2
        echo "Task $i completed"
    ) &
done

# すべてのバックグラウンドジョブの完了を待つ
wait
echo "All tasks completed"
```

### エラーハンドリング

```bash
#!/bin/bash

for file in *.txt
do
    if [ ! -f "$file" ]; then
        echo "Warning: $file not found"
        continue
    fi
    
    if ! cp "$file" "/backup/$file"; then
        echo "Error: Failed to copy $file"
        exit 1
    fi
    
    echo "Copied: $file"
done
```

### 進捗表示

```bash
#!/bin/bash

files=(*.txt)
total=${#files[@]}
current=0

for file in "${files[@]}"
do
    current=$((current + 1))
    percent=$((current * 100 / total))
    echo "Processing [$current/$total] ($percent%): $file"
    # 処理
    sleep 0.5
done
```

## ループのパフォーマンス最適化

### 不要なコマンド呼び出しを避ける

```bash
# 悪い例: 毎回dateコマンドを呼び出す
for i in {1..1000}
do
    timestamp=$(date +%s)
    echo "$timestamp: Processing $i"
done

# 良い例: ループの外で1回だけ呼び出す
timestamp=$(date +%s)
for i in {1..1000}
do
    echo "$timestamp: Processing $i"
done
```

### サブシェルの回避

```bash
# 悪い例: パイプでサブシェルが作られる
cat file.txt | while read line
do
    count=$((count + 1))
done
echo $count  # 0（サブシェル内の変更が反映されない）

# 良い例: リダイレクトを使用
count=0
while read line
do
    count=$((count + 1))
done < file.txt
echo $count  # 正しい値
```
