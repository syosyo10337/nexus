# 特殊変数

[https://qiita.com/a_yasui/items/ec4f75b300410af8958d](https://qiita.com/a_yasui/items/ec4f75b300410af8958d)

| 特殊変数 | 説明 |
|----------|------|
|`$?`|直前に実行したコマンドや関数の**終了ステータス**（成否を示す数値）を格納する特殊な変数です。|
|`$!`|直前に実行したコマンドや関数の**プロセスID**を格納する特殊な変数です。|
|`$#`|直前に実行したコマンドや関数の**引数の数**を格納する特殊な変数です。|
|`$@`|直前に実行したコマンドや関数の**引数のリスト**を格納する特殊な変数です。|
|`$*`|直前に実行したコマンドや関数の**引数のリスト**を格納する特殊な変数です。|
