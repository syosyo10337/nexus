# 導入

### 1. **代表的な圧縮形式**

なんとなく、圧縮形式を知る。`.zip` の拡張子のファイル以外にも圧縮ファイルはあるのだよ。。

- **tar**: アーカイブ形式で、複数のファイルやディレクトリを1つのファイルにまとめるもの（圧縮はしない）。通常、他の圧縮形式（gzipやbzip2）と組み合わせて使用されます。
    
    - 例: `.tar`, `.tar.gz`, `.tar.bz2`
    

- **gzip**: ファイルを圧縮する形式。圧縮率は高く、速度も速い。
    
    - 例: `.gz`
    

- **bzip2**: gzipよりも高い圧縮率を提供しますが、やや遅い。
    
    - 例: `.bz2`
    

- **xz**: 圧縮率が高く、最近のLinuxディストリビューションでよく使用されます。
    
    - 例: `.xz`, `.tar.xz`
    

- **zip**: Windowsでもよく使われる圧縮形式。ファイルを圧縮しながらアーカイブします。
    
    - 例: `.zip`
    

- **7z**: 高圧縮率で知られる形式。Windowsの7-Zipでよく利用されます。
    
    - 例: `.7z`
    

プログラムで触っててたまに見るのはtarコマンドな気がするね。たまにunzipもするか。ダブルクリックで完了することも多いけれども。

---

### 2. **各形式の解凍・圧縮コマンド**

### **tar アーカイブ**

- **解凍:**
    
    ```Shell
    tar -xvf archive.tar
    ```
    
    - `x`: 解凍
    
    - `v`: 詳細表示
    
    - `f`: 対象ファイルを指定
    

- **圧縮:**
    
    ```Shell
    tar -cvf archive.tar file1 file2 dir1
    ```
    
    - `c`: 圧縮（アーカイブ作成）
    

### **tar + gzip (.tar.gz または .tgz)**

- **解凍:**
    
    ```Shell
    tar -xvzf archive.tar.gz
    ```
    
    - `z`: gzip形式で圧縮されたtarファイルを扱う
    

- **圧縮:**
    
    ```Shell
    tar -cvzf archive.tar.gz file1 file2 dir1
    ```
    

### **tar + bzip2 (.tar.bz2)**

- **解凍:**
    
    ```Shell
    tar -xvjf archive.tar.bz2
    ```
    
    - `j`: bzip2形式で圧縮されたtarファイルを扱う
    

- **圧縮:**
    
    ```Shell
    tar -cvjf archive.tar.bz2 file1 file2 dir1
    ```
    

### **tar + xz (.tar.xz)**

- **解凍:**
    
    ```Shell
    tar -xvJf archive.tar.xz
    ```
    
    - `J`: xz形式で圧縮されたtarファイルを扱う
    

- **圧縮:**
    
    ```Shell
    tar -cvJf archive.tar.xz file1 file2 dir1
    ```
    

---

### **gzip (.gz)**

- **解凍:**
    
    ```Shell
    gunzip file.gz
    ```
    

- **圧縮:**
    
    ```Shell
    gzip file
    ```
    

### **bzip2 (.bz2)**

- **解凍:**
    
    ```Shell
    bunzip2 file.bz2
    ```
    

- **圧縮:**
    
    ```Shell
    bzip2 file
    ```
    

### **xz (.xz)**

- **解凍:**
    
    ```Shell
    unxz file.xz
    ```
    

- **圧縮:**
    
    ```Shell
    xz file
    ```
    

---

### **zip**

- **解凍:**
    
    ```Shell
    unzip archive.zip
    ```
    

- **圧縮:**
    
    ```Shell
    zip archive.zip file1 file2 dir1
    ```
    

---

### **7z (.7z)**

- **解凍:**
    
    ```Shell
    7z x archive.7z
    ```
    
    （7zはインストールが必要: `sudo apt install p7zip-full`）
    

- **圧縮:**
    
    ```Shell
    7z a archive.7z file1 file2 dir1
    ```
    

---

### 3. **ファイル形式を確認する**

圧縮形式が分からない場合、`file`コマンドで確認できます。

```Shell
file archive.tar.gz
```

例:

```Plain
archive.tar.gz: gzip compressed data
```

---

### 4. **便利なTips**

- **圧縮ファイルを作成しながら一覧を表示したい:**
    
    ```Shell
    tar -cvzf archive.tar.gz file1 file2 dir1
    ```
    
    このコマンドで何が圧縮されているかが確認できます。
    

- **ディレクトリ全体を圧縮・解凍したい:**  
    圧縮時や解凍時にディレクトリ名をそのまま指定可能です。
