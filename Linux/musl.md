 

# musl

**Linuxベースのオペレーティングシステム（OS）向けの、軽量でセキュアかつ標準に準拠した標準Cライブラリ**

**主な特徴**

- **標準Cライブラリの実装:**C言語で記述されたプログラムが利用する標準的な関数群（入出力、文字列操作など）を提供する役割を担います。

- **軽量かつセキュア:**glibc（GNU C Library）に代わる選択肢として、より小さいコードベースと高いセキュリティを追求しています。

- **静的リンクに最適化:**アプリケーションを静的にリンクすることで、依存するライブラリがなくても単一の実行可能ファイルを生成できます。これは、Alpine Linuxなどの環境でアプリケーションをデプロイする際に役立ちます。
