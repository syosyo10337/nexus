 

# ユーザの権限を移譲する。

---

1. ユーザを作成する

```Bash
$ sudo adduser <ユーザ名>

$ sudo passwd <ユーザ名>
# インストラクションに続いて、パスを設定
```

2. スーパーユーザと呼ばれる管理者権限を持つユーザの権限範囲を編集する

```Bash
$ sudo visudo
```

コマンドによって展開されるVimの中で、rootの下にあるはずの、追加したユーザに対して、 `ALL=(ALL) ALL`を追加する

```Bash
# 権限を編集ファイル上

root    ALL=(ALL)       ALL
<追加したユーザ>  ALL=(ALL)       ALL
```

3. ユーザを実際に切り替える

```Bash
sudo su <ユーザ名>
```

4. ローカルに`ssh-keygen`などで鍵を用意して、sshでの接続の準備をする。

5. 実際に接続が確認できたらば、rootを削除する。(ec2-userという名前で攻撃対象にならないように)

```Bash
$ sudo userdel -r ec2-user
```
