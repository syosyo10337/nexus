# 数値の単位

## px:

固定された単位で、画面計測上の最小単位です。絶対値の単位として扱われますが、デバイスの種類によって長さが異なるため、印刷物ではない画面表示には px を使うように推奨されています

## em:

(font-sizeプロパティにおいては)親要素のfont-sizeに対する相対値として定義され、それ以外のプロパティにおけるem単位は、現在の要素のfont-sizeに対する相対値として定義されます。  
==仮に親要素に font-size が設定されていない場合、HTML の根ノードに向かって探索されます。根ノードでも font-size が設定されていない場合、ブラウザのデフォルト値である 16px が基準値となります。==

## rem:

HTML 文書の根ノードの font-size プロパティの値を基準とした相対値として定義されます。(根ノードで font-size が設定されていない場合、ブラウザのデフォルト値である16px が基準値となります。)  
emとは異なり、htmlタグのfont-sizeプロパティの値を基準とするため、予期せぬ複合効果を避けることができる。また、font-sizeプロパティ以外も根ノードの font-size プロパティの値が基準値となります。

## %:

他の値を基準とした値で、親要素を基準とした相対値として使用されることが多いです。  
margin, padding, font-size, height, width 等さまざまなプロパティで使用されます。

## vh/vw:

ビューポート(閲覧中の文書で現在見えている領域)のサイズを100として計算する単位。

h → height/w → witdh  
e.g)  
ビューポートの高さが 720px の場合、1vh は 7.2px
