# width/height: コンテンツの横幅/高さ指定

(box-sizing: content-boxの時)要素自体のサイズは, paddingとmarginを足したものになる点に注意する。

- box-sizing: border-box;  
    コンテンツの領域(width/heightで指定できる範囲)をpaddingとborderを含めて計算する

- overflow: --指定した領域から溢れたコンテンツを扱う(値:hidden-溢れた分は隠す,scroll-スクロールしてみれるようになる。auto-ブラウザに任せる。)

# borderについて

## border:

(一括指定プロパティ) width style color(順不同);

- border-top/-right/-bottom/-left:と1方向だけに指定できる。

### border-width:

- ボーダーの線の幅を指定(単位: px, thick, medium, thin)

- border-left-widthのように一辺での指定も可能

### border-style:

種類(e.g.solid, dashed, dotted, groove)

### border-color: 色/一辺での指定も可能

### border-radius: (px,%,em)ボーターの角を丸める。

# paddingについて

### padding: 余白を入れる(default:0)

(値 top right bottom left)

- 入れた値が1つの時(４方向に適応)

- 入れた値が2つの時: (top+bottom) (right+left);

- 入れた値が3つの時: (top) (right+left) (bottom);  
    /*1方向だけに入れたい場合は  
    padding-left/-right/-top/-bottom  
    と指定できる。 */

- paddingを入れる際の単位はemが推奨(現在の子子要素の文字サイズから算出されるため。) %は親要素に影響されるから

# marginについて

### margin: 枠線(border)の外側の余白

_(値:auto;)--方向を指定すると、指定方向に自動で余白を作る。左右または上下でまとめてautoとすると、左右または上下の中央揃えをしてくれる。  
/_  
その際には、左右であればセレクタのwidth,  
上下であればheightを必ず指定する.またこれはブロック要素に対する中央揃えに使う*/

- marginの相殺--垂直方向でmarginが重なると、小さい方が打ち消される。

- +の値と-の値の場合には、単純にマイナス分を差し引いた値がmarginとして入ります！

- marginにマイナスの値を取ることができる。  
    この時、ボックスの値は常に以下の計算式によって作成される。  
    e.g)  
    親要素の幅 = margin + border + padding + コンテンツ
