# tsvファイルの読み込み

tab separated values fileを読み込む。

```Python
df = read.csv('<file_path>', sep = '\t')
```

デフォルトでは`sep = ‘,’`の`read_csv()`関数の区切り文字をtabに指定する。

# データフレームのmerge

`merge()` メソッドを使うことで、二つの異なるデータフレームを結合key(外部key)を指定して結合することができる。

## 内部結合  
`df1.merge(df2, how = ‘inner’, on = ‘<df1の外部key>’)`

```Python
e.g.)
df2 = df.merge(wheater, how = 'inner', on = 'wheater_id')
```

## 左外部結合  
`df1.merge(df2, how = ‘left’, on = ‘<df1の外部key>’)`

```Python
e.g.)
temp.T.loc[199:201]
df2[df2['dteday'] == '2011-07-20'] # df2にはあるんよ


# temp_tranposedを子として、df2に左外部結合をおこなう。結合key 'dteday'
temp_tranposed = temp.T 
df3 = df2.merge(temp_tranposed, how = 'left', on = 'dteday')

df3[df2['dteday'] == '2011-07-20']
```
