 

# 平均絶対誤差/平均二乗誤差/決定係数

[決定係数(R-squared: R*2)](#102b8a0a-0f7e-49e8-9111-efaf20f2f44e)

scoreメソッドを用いて,学習されたモデルを評価しようとするが、この値は正解率ではない。

回帰分析における、評価指標として次の二つが考えられる

- **平均二乗誤差(mean squared error: MSE)  
    **差分を2乗したデータでの平均値

- **平均絶対誤差(mean absolute error : MAE)  
    **差分の絶対値の平均値

## **平均絶対誤差**

sklearnのmetricsモジュールを引っ張り出して、MAEを算出してみる。

```Python
from sklearn.metrics import mean_absolute_error

pred = model.predict(x_test)


mean_absolute_error(y_pred = pred, y_true = y_test)
#-> 277.1223696408626
```

### `mean_absolute_error(y_pred = <予測されるデータ>, y_true = <用意した正解データ>)`

この時、目的に対しての誤差の大小が数値自体からは判断しきれないため、正解データに対する専門知識が必要になります。

# 決定係数(R-squared: R*2)

これは0 ~ 1の間の値を取り、値が大きくなるほど予測値と実測値の誤差が少ない回帰式であることを意味する指標。

正解データの意味に、依存しないため、純粋に数値として評価できる。==一般に0.8以上が予測性能の高い計算式と呼ばれる==

`sklearn.linear_model.LinearReggression`における`score`メソッドは、この決定係数を返すものになっている。
