その前に、modelの中身を理解していこう。

# 分岐条件の列(特徴量)

```Python
model.tree_.feature

#-> array([ 3, -2,  3, -2, -2], dtype=int64)
```

それぞれの値は、featureつまり特徴量のcolmun_indexを示している。値が**-2になっているものは分岐が終了していることを表す。**

よって、この場合には、

[![](ML/content/Attachments/%E7%84%A1%E9%A1%8C-2023-02-09-2116%201.svg)](%E6%B1%BA%E5%AE%9A%E6%9C%A8%E3%81%AE%E5%9B%B3%E3%81%AE%E4%BD%9C%E6%88%90/%25E7%2584%25A1%25E9%25A1%258C-2023-02-09-2116.svg)

ということがわかる。[3]は’花弁の幅’についてでした。

# 分岐条件のしきい値

```Python
model.tree_.threshold

#->array([ 0.29499999, -2.        ,  0.69      , -2.        , -2.        ])
```

scikit-learnにおいては、

==“<分岐条件に用いられる列の値>” ≤ しきい値==

の書式で、比較させている。.thresholdで出てくる値はここでいう

==しきい値==

にあたる部分です。つまり、ここまでの話だと,

[![](ML/content/Attachments/%E7%84%A1%E9%A1%8C\(2\)-2023-02-09-2116.svg)](%E6%B1%BA%E5%AE%9A%E6%9C%A8%E3%81%AE%E5%9B%B3%E3%81%AE%E4%BD%9C%E6%88%90/%25E7%2584%25A1%25E9%25A1%258C\(2\)-2023-02-09-2116.svg)

ということがわかる

# 末端ノードの種類を紐づける

```Python
e.g.)

# 末端のノードつまり、判定された時の値から、を表示value[index]
print(model.tree_.value[1])
print(model.tree_.value[3])
print(model.tree_.value[4])

#->
# [[31.  0.  0.]]
# [[ 0. 36.  7.]]
# [[ 0.  1. 30.]]
```

実際の正解ラベルの種類と紐づけたいのですが、ここでこの配列が意味するのは、グループのことです。

グループの対応をしるために

```Python
e.g.)

model.classes_

#->
# array(['Iris-setosa', 'Iris-versicolor', 'Iris-virginica'], dtype=object)
```

これによって、

末端ノードの[3]では、

'Iris-setosa’のデータが0,

'Iris-versicolorのデータが36,

'Iris-virginica'のデータが7

が到達したことを示している。

このことから

# [[31. 0. 0.]] → 'Iris-setosa’

# [[ 0. 36. 7.]] → 'Iris-versicolor  
# [[ 0. 1. 30.]] → 'Iris-virginica'

と判定されていることがわかる。

よって、作成されたモデルは

[![](ML/content/Attachments/%E7%84%A1%E9%A1%8C\(3\)-2023-02-09-2116.svg)](%E6%B1%BA%E5%AE%9A%E6%9C%A8%E3%81%AE%E5%9B%B3%E3%81%AE%E4%BD%9C%E6%88%90/%25E7%2584%25A1%25E9%25A1%258C\(3\)-2023-02-09-2116.svg)

## [使ったメソッドまとめ]

- model.tree_.feature  
    特徴値を取得

- model.tree_.threshold  
    しきい値を取得

- model.tree_.value[n]  
    到達したデータ群を取得

- mode.classes_  
    正解のラベルのグループの内訳を表示

✅

classes_/tree_　などと_が記される属性では、モデルによって学習済みの値を参照するときの目印になる。

## 描画の一例

```Python
from sklearn.tree import plot_tree


plot_tree(model, filled = True)
```
