# 設定関連

Machine Learning

- Jupyterを使うらしい

- 拡張子は .ipynb  
    [https://ai-inter1.com/jupyter-notebook/](https://ai-inter1.com/jupyter-notebook/)

[基本操作]  
・セルごとに実行ができます。

・セルの最後に記述した値（変数）のみ、print文なしで出力できます。

・[Ctrl]+[Enter]で実行できます。

・[Shift]+[Enter]で実行して次のセルを追加します。

・その他、VSCodeのショートカットキーも使用できます。

docker compose upして token付きでサーバに接続(VScodeからでもok
