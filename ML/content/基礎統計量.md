## [代表値]

### 中央値 (median)

データの個数を最大値から数えてちょうど半分のところが中央値です。データの個数が奇数個ならばちょうど半分のところであり、偶数個ならば前後の数を足して2で割った値です。中央値は外れ値の影響を受けにくいのが特徴です。

cf)第３四分位数や、第1四分位数もある。

### 分散

データのばらつきを表す

各データの偏差を求めて、その偏差それぞれを２乗した値の平均をとる。

s2=1n∑i=1n(xi−xˉ)2s^2 = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})^2s2=n1​i=1∑n​(xi​−xˉ)2

### 標準偏差(Standard diviation:SD)

分散を表現するために、元データに２乗する形をとった分を平方根することで、データの単位を本来のデータと同じ単位にするための表現。

s=s2s = \sqrt{s^2}s=s2​

### 共分散  
(covariance)

「国語の点数」と「数学の点数」のような「二組の対応するデータ」の間の関係を表す数値

- **求め方**

(xの偏差)(yの偏差)の平均値

sxy=1n∑i=1n(xi−xˉ)(yi−yˉ)s_{xy} = \frac{1}{n}\sum_{i=1}^{n}(x_i -\bar{x})(y_i - \bar{y})sxy​=n1​i=1∑n​(xi​−xˉ)(yi​−yˉ​)

- 共分散の値の意味
    
    **共分散が大きい（正）→** _**X**_ **が大きいとき** _**Y**_ **も大きい傾向がある**
    
    **共分散が 0 に近い→** _**X**_ **と** _**Y**_ **にあまり関係はない**
    
    **共分散が小さい（負）→** _**X**_ **が大きいとき** _**Y**_ **は小さい傾向がある**
    

と言える。

### 相関係数

- 求め方

(xとyの共分散)を (xの標準偏差とyの標準偏差の積)で割った値のこと。

r=1n∑i=1n(xi−xˉ)(yi−yˉ)1n∑i=1n(xi−xˉ)21n∑i=1n(yi−yˉ)2=sxysxsyr = \frac{\frac{1}{n}\sum_{i=1}^{n}(x_{i}-\bar{x})(y_{i}-\bar{y})}{\sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_{i}-\bar{x})^{2}}\sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_{i}-\bar{y})^{2}}} = \frac{s_{xy}}{s_{x}s_{y}}r=n1​∑i=1n​(xi​−xˉ)2​n1​∑i=1n​(yi​−yˉ​)2​n1​∑i=1n​(xi​−xˉ)(yi​−yˉ​)​=sx​sy​sxy​​

- 値の持つ意味
    
    相関係数rは -1 ≤ r ≤ 1の値を取り、
    
    - -0.3 < r < 0.3**：相関がない**
    
    - (+-)0.3 ≤ r < (+-)0.5 : 非常に弱い相関
    
    - (+-)0.5 ≤ r < (+-)0.7 : 相関がある。
    
    - (+-)0.7 ≤ r < (+-)0.9：強い相関
    
    - (+-)0.9 ≤ r ：非常に強い相関
    
    と言われている、相関係数が正の値のときには**正の相関.**
    
    負の時は**負の相関がある。**と呼ばれている。
