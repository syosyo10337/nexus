# 代表値の計算

---

## `DataFrame.mean()`

数値列の、各平均値を計算する

(文字列columnは自動で除外し集計する。)

☝

==将来的に、自動で数値だけを算出する仕様をなくしますよ==

The default value of numeric_only in DataFrame.mean is deprecated. In a future version, it will default to False. In addition, specifying 'numeric_only=None' is deprecated. Select only valid columns or specify the value of numeric_only to silence this warning.  
df.mean()  
  
  
後述の方法を使って、数値列だけに使用できるように特定してあげましょう。

### 特定の列だけを計算する

```Python
e.g.)
df['がく片長さ'].mean()

#->
0.42087837837837844
```

### その他代表値を算出メソッド

e.g)

- df.var() — 分散

- df.mean() — 平均

- df.std() — 標準偏差

- df.median() — 中央値

- df.sum() — 合計

- df.max() — 最大値

- df.min() — 最小値

- df.mode() — 最頻値

🚨

modeについて、最頻値が複数出る場合に関して, series型の値で返るので[]インデックスを指定すること。

# `df.describe()`

データフレームに対して使うことで、カラムごとの代表値の種類をindexとする一覧表(dataframe)を表示する

- `exclude = ‘number’`オプション  
    このオプションを指定すると数値ではない、値のみの代表値計算をしてくれる
