## `DataFrame.drop(index, axis=0)`

indexを指定して、特定のrowを削除したデータフレームを返す。

drop関数を使う

インデックスをリストで指定することで、複数削除できる

```Python
e.g.)
a = test.drop(0, axis= 0)
#-> a
#test自体は変わっていない。inplaceオプションで破壊的に変更することも可能。
```

## `DataFrame.drop(col_name, axis=1)`

column名を指定して、特定のcolumnsを削除したデータフレームを返す。

==カラム名をリストで指定することで、複数削除できる==

- inplaceオプションで、破壊的変更も可能

pd.drop

```Python
new_x = x_tmp.drop('Sex', axis =1)
```
