### `unique()`

Seriesまたは、numpy.array型に対して使用することで、重複した値を取り除いた値を返してくれる。(1次元の値に対して使用する。)

戻り値は、array型(numpy)

```Python
df['種類'].unique()
 
#-> 
array([
 'Iris-setosa', 
 'Iris-versicolor', 
 'Iris-virginica'
 ], dtype=object
)
```

☝

array型は通常のリストのようにインデックスで参照することができる  
e.g.)  
```  
category = df[’種類’]  
category[0] #→  
```

## `value_counts()`

uniqueメソッドで算出できる、ある列のデータの種類ごとにカウントアップしてくれるメソッド

```Python
df['種類'].value_counts()

#->
Iris-setosa        50
Iris-versicolor    50
Iris-virginica     50
Name: 種類, dtype: int64
```
