# 線形補間

時系列データに代表するような、相互に依存するデータにおいて、欠損値を補間する方法のひとつ

以下のようなデータの間を補間すること。

[![](ML/content/Attachments/Screenshot_2023-02-16_at_21.24.39.png)](%E6%AC%A0%E6%90%8D%E5%80%A4%E3%81%AE%E5%87%A6%E7%90%86/Screenshot_2023-02-16_at_21.24.39.png)

実際に補間してみる

```Python
df3['atemp'] = df3['atemp'].astype(float)
df3['atemp']  = df3['atemp'].interpolate()

df3.loc[220:240, 'atemp'].plot(kind = 'line')
```

## `df[’col’].astype(<data_type>)`

引数にとったデータ型(e.g float)などにデータ型を変換する

## `df[’col’].interpolate()`

欠損値を線形補間したdfを返す
