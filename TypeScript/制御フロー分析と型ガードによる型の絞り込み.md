# 制御フロー分析と型ガードによる型の絞り込み

## **制御フロー分析**

これのこと。これによって、monthがstringの時にだけ、実行できるメソッドを指定してもエラーにならない。

```TypeScript
function showMonth(month: string | number) {
  if (typeof month === "string") {
    console.log(month.padStart(2, "0"));
  }
}
```

```TypeScript
function showMonth(month: string | number) {
  if (typeof month === "string") {
    console.log(month.padStart(2, "0"));
  }
  console.log(month.toFixed());
}



// 早期リターンするとこれが成立する。
function showMonth(month: string | number) {
  if (typeof month === "string") {
    console.log(month.padStart(2, "0"));
    return;
  }
  console.log(month.toFixed());
}
```

## 型ガード

typeof /**instanceof**

特定のクラスのインスタンスであることを確認したいのであれば。 instanceofを使う。

## in

[](https://typescriptbook.jp/reference/statements/control-flow-analysis-and-type-guard#in)

特定のクラスのインスタンスであることを明示せず、`in`演算子でオブジェクトが特定のプロパティを持つかを判定する型ガードを書くことで型を絞り込むこともできます。

```TypeScript
interface Wizard {
  castMagic(): void;
}
interface SwordMan {
  slashSword(): void;
}
 
function attack(player: Wizard | SwordMan) {
  if ("castMagic" in player) {
    player.castMagic();
  } else {
    player.slashSword();
  }
}
```

### user-defined type guard / type guarding function

```TypeScript
function isWizard(player: Player): player is Wizard {
  return "castMagic" in player;
}
// trueの時にplayerはWizard型であることを返す関数

function attack(player: Wizard | SwordMan) {
  if (isWizard(player)) {
    player.castMagic();
  } else {
    player.slashSword();
  }
}
```

### 型ガードの変数代入

* TypeScript4.4 以降

```TypeScript
function getMonth(date: string | Date) {
  const isDate = date instanceof Date;
  if (isDate) {
    console.log(date.getMonth() + 1);
  }
}
```
