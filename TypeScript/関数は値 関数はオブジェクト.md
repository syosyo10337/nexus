 

# 関数は値/関数はオブジェクト

JavaScriptの関数は値です。つまり、PHPのような他の言語と比べると特別扱いの度合いが少ないです。たとえば、関数を変数に代入することができます。

また、JavaScriptでは定義済みの関数と同じ名前の関数を宣言することができます。これはエラーにはなりません。これは実質、再代入のような振る舞いになります。

```TypeScript
function hello() {
  return "HELLO";
}
 
// これは二度目の関数宣言ですが、実質的には再代入です
function hello() {
  return "KONNICHIWA";
}
 
hello();
```

JSで気をつけるには、もうconst/関数式を組み合わせるほかありません。TSはコンパイルエラーにしてくれます。

**関数は値なので、関数名のスコープも変数と同じようにスコープの概念があります。たとえば、関数スコープの中で定義された関数は、そのローカルスコープでのみ使うことができます。**

つまり、関数内で定義されたローカルスコープの変数を関数の外で参照できないのと同様に、関数内の関数もローカルスコープを作り、関数外から参照することはできない。

# JSの関数はオブジェクトでもあります

```TypeScript
function hello() {
  return "Hello World";
}
 
hello.prop = 123;
```
