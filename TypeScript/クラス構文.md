 

# クラス構文

```JavaScript
class Foo {
    x: number;
    constructor(x:number) {
        this.x = x;
    }
}
```

TypeScriptだと省略構文が使えるよ

```TypeScript
class Foo {
    constructor(public x:number) {
    }
}
```

IIFEは{}()で即時よみ出しすること

TypeScriptのクラスはIIFEに包含されている。まだ

TypeScriptは、IIFEによってベースクラスを変数`_super`として補足（キャプチャ）できるようにしています:

extendsの中身

```TypeScript
var __extends = this.__extends || function (d, b) {
    for (var p in b) if (b.hasOwnProperty(p)) d[p] = b[p];
    function __() { this.constructor = d; }
    __.prototype = b.prototype;
    d.prototype = new __();
};
```

これが、JS(TS)がクラス構文をサポートできている理由
